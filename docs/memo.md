# Todo App

## 目的

実務を意識した学習用Todoアプリを作成する。

---

## 要件

### 必須

- Todo追加
- Todo一覧表示
- Todo削除
- 完了/未完了切替

### 追加機能

- 編集
- 並び替え
- ピックアップ
- 検索

---

## 開発手順

### Phase1 UI作成

HTML/CSS

### Phase2 ロジック実装

JavaScript

### Phase3 保存

LocalStorage

### Phase4 API化

Node.js

### Phase5 DB

MariaDB

fetch
